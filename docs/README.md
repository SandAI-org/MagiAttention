## Instructions to build docs locally

#### Step1: get into docs directory

```bash
cd docs
```

#### Step2: install requirements

```bash
pip install -r requirements.txt
```

#### Step3: generate docs

```bash
make clean
make html
```

#### Step4: Download build/html and view it in your browser
